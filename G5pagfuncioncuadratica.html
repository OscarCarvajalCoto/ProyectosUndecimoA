Función Cuadrática

Descripción

La función cuadrática es una función polinómica de segundo grado, cuya gráfica es una parábola. Se representa como:

f(x) = ax² + bx + c, donde a ≠ 0

Donde:

  • x: preimagen
  • f(x): imagen
  • a, b, c: coeficientes reales, con a diferente de cero

Dominio y Ámbito:

Dominio: todos los números reales (ℝ)

Ámbito: depende del valor de a:

Eje de simetría

x = -b/2a

Vértice:

Punto máximo o mínimo, según el signo de a, calculado como:

  • h = -b / 2a
  • x = (-b ± √(b² - 4ac)) / 2a

Intersecciones:

  • Con el eje y: (0, c)
  • Con el eje x: se obtiene resolviendo (ax² + bx + c = 0) (puede tener 0, 1 o 2 soluciones reales)
