Función Logarítmica

Descripción

Una función logarítmica es la inversa de una función exponencial. Representa el exponente al que se debe elevar la base para obtener cierto número.

La función logarítmica se representa como:

f(x) = logₐ(x), donde a > 0 y a ≠ 1.

Donde:

  • a: base
  • x: argumento

Dominio y Ámbito:

Dominio: ]0, más infinito[

Ámbito: todos los números reales (ℝ)

Características:

  • Asíntota: vertical en (x = 0)
  • Intersección con eje x: (1, 0)
  • No interseca el eje y
  • Es biyectiva, su inversa es la función exponencial.

Comportamiento según la base:

  • Si (a > 1) → función creciente
  • Si \(0 < a < 1\) → función decreciente
